# Start the Localhost App

Use these steps to run English Connectors locally.

## Prerequisites
- Node.js installed
- Dependencies installed with `npm install`

## Start the app
1. Open a terminal in the project root:
   `c:\Programming\JavaScript\English Connectors`
2. Start the server:
   `npm start`
3. Open the app in your browser:
   `http://localhost:3000`

## Stop the app
- In the terminal where the server is running, press `Ctrl + C`.

## Notes
- If you see build or runtime errors, run `npm run build` first to verify the app compiles.
- The app uses the built server entrypoint from `dist/server/server/index.js` when started.
